Case1 并发多会话场景 Token 消耗测试

操作：多终端并行拉起openclaw chat，多会话同时下发【Linux 运维命令查询】基线任务；
统计目标：多并发汇总总 Token、单会话平均 Token，和单人基线对比；
预期：单会话 Token 消耗贴近单人基线，并发总量 = 会话数 × 单基准值，无异常超额吞 Token；
实测备注：OpenAI 无 API Key，所有会话 401 报错无法运行，无真实消耗数据，依托标准基线完成理论比对归档。
