# Day6 多场景Token消耗专项测试用例
## 测试说明
1. 测试基准：复用 Day3 单任务基线（Linux 运维搜索任务、Python 代码编写任务标准 Token 基线值）；
2. 测试入口：当前 OpenClaw 2026.5.7 版本无openclaw run/agents run等非交互任务指令，仅可使用openclaw chat交互式会话开展测试；
3. 环境约束：原生客户端无有效 OpenAI API 密钥，模型调用触发 401 鉴权失败，无法通过原生会话采集数据。本次使用 Python 脚本直连模型接口完成全场景实测，成功采集真实 Token 消耗、响应耗时数据。
